Nginx 静态化部署说明

本项目的活动详情页通过 FreeMarker 自动生成静态 HTML，适合交给 Nginx 直接发布，降低动态接口压力并满足课程静态化评分点。

静态页生成目录

默认目录：generated-static/activities

活动发布、报名变化、活动截止关闭后，后台会自动刷新对应的静态页文件。

推荐 Nginx 配置

server {
    listen 8088;
    server_name localhost;

    location /activities-static/ {
        alias /Users/xujunming/idea-java/网站架构项目大作业/Campus Service ManagerAI/generated-static/activities/;
        index index.html;
        autoindex on;
        charset utf-8;
    }
}

演示建议

  • 先在后台创建或修改一场活动，保存后会自动生成静态页。
  • 在学生端完成报名或取消报名，再回到静态目录查看对应 HTML 更新时间。
  • 等待活动结束或手动调整时间，后台定时任务会自动关闭活动并刷新静态页。
