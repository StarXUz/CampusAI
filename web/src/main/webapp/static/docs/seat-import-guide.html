Seat Import Guide

座位导入说明

用于后台管理端批量导入图书馆 / 自习室座位。建议先创建自习室，再下载 Excel 模板填写数据，最后回到“自习室管理”页面执行导入。

导入步骤

  1. 先在后台新增自习室，确保存在有效的 study_room_id。
  2. 点击“下载 Excel 模板”，按模板中的字段顺序填写。
  3. 每一行代表 1 个座位，支持不同楼层和不同区域混合导入。
  4. 回到后台“自习室管理”页上传 Excel 文件，导入成功后会即时写入数据库。

字段说明

字段 说明 示例
study_room_id 自习室 ID，必须已存在于系统中。 1
seat_code 座位编号，建议按楼层 + 分区 + 序号统一编码。 A101 / 3F-C-018
floor_no 楼层编号，学生端可按楼层切换座位布局。 1 / 2 / 3
zone_name 区域名称，会在学生端用于分区展示。 临窗静音区 / 开放学习区

推荐编码规则

  • 一楼临窗区：1F-W-001 到 1F-W-060
  • 二楼安静区：2F-Q-001 到 2F-Q-080
  • 三楼开放区：3F-O-001 到 3F-O-120

填写注意事项

  • 不要修改首行表头名称。
  • 空白行会自动跳过，但字段为空的非空行会报错。
  • 同一自习室建议避免重复的 seat_code。
  • 导入前先确认楼层和区域命名与学生端展示一致。
